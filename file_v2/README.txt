This is the explanation of the csv in this folder

dropped_columns_matrix.csv : columns with more that 30% of NaN values are excluded. The column name, the ratio of NaN values, data type, and the values are listed in the spreadsheet.
dropped_rows.csv : rows with more than 10% of NaN values are exclued. (50 rows)
data_preprocessed_v2.csv : data after this cleansing is reported.
